# LangChain Practice Workspace

A hands-on learning repository for building AI applications with Python, LangChain, and modern LLM tools. This project combines small scripts, notebooks, and course materials to help you explore prompt engineering, retrieval, agents, memory, and multimodal workflows.

## ✨ What’s inside

- Practical examples using LangChain and related libraries
- Notebook-based experiments for learning and prototyping
- Small Python scripts for quick testing and demos
- Course content focused on agentic AI and RAG-style workflows

## 🧭 Project structure

- `main.py` — simple entry point for the workspace
- `course/` — course material and example projects
- `practise/` — notebook experiments and practice files
- `pyproject.toml` — project dependencies and metadata

## 🚀 Getting started

### 1. Install dependencies

If you use `uv`:

```bash
uv sync
```

Or with `pip`:

```bash
pip install -r requirements.txt
```

> If you do not have a `requirements.txt` file yet, install the packages listed in `pyproject.toml` directly with your preferred package manager.

### 2. Run the project

```bash
python main.py
```

### 3. Explore notebooks

Open the files in the `practise/` and `course/` folders with Jupyter or VS Code notebooks.

## 🛠️ Recommended tools

- Python 3.12+
- VS Code
- Jupyter Notebook / JupyterLab
- `uv` for dependency management

## 📚 Learning goals

This repository is designed to help you understand:

- LLM calling basics
- Prompt design and chaining
- Vector stores and retrieval
- Agents and tool use
- Memory in conversational AI
- Evaluation and guardrails

## 💡 Notes

- Some examples may require API keys or external services.
- Keep your environment variables in a local `.env` file and avoid committing secrets.
